\doxysubsubsection{Detailed Description}
Initialize the LFXO oscillator. 

Configure the low frequency crystal oscillator using settings in the configuration header {\ttfamily \doxylink{sl__device__init__lfxo__config_8h}{sl\+\_\+device\+\_\+init\+\_\+lfxo\+\_\+config.\+h}}.

The precision of the oscillator can be indicated using the configuration option {\ttfamily SL\+\_\+\+DEVICE\+\_\+\+INIT\+\_\+\+LFXO\+\_\+\+PRECISION}. This is used by certain wireless stacks to optimize wakeup time for scheduled events (such as Bluetooth advertising).

If using Simplicity Studio, this configuration header is also configurable through the Project Configurator by selecting the "{}\+Device Init\+: LFXO"{} configuration component.
