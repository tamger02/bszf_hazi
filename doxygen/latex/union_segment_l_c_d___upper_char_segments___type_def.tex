\doxysubsection{Detailed Description}
Typedef to store the segment data of a 7-\/segment digit. 

The typedef uses an 8 bit integer to store the data. As we need only 7 segments, the MSB is unused. The bits are mapped to the segments in the following way\+:

--- 0 (a) --- \texorpdfstring{$\vert$}{|} \texorpdfstring{$\vert$}{|} \texorpdfstring{$\vert$}{|}5 (f) \texorpdfstring{$\vert$}{|}1 (b) \texorpdfstring{$\vert$}{|} \texorpdfstring{$\vert$}{|} --- 6 (g) --- \texorpdfstring{$\vert$}{|} \texorpdfstring{$\vert$}{|} \texorpdfstring{$\vert$}{|}4 (e) \texorpdfstring{$\vert$}{|}2 (c) \texorpdfstring{$\vert$}{|} \texorpdfstring{$\vert$}{|} --- 3 (d) ---

The bits can be accessed in two ways\+:


\begin{DoxyEnumerate}
\item Using a single, 8 bit value (raw).
\item Accessing individual bits using bit fields (a..g). 
\end{DoxyEnumerate}
