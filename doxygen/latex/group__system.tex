\doxysubsection{Detailed Description}
System API. 

System Initialization and Action Processing.

This module contains functions to read information such as RAM and Flash size, device unique ID, chip revision, family, and part number from DEVINFO and SCB blocks. Functions to configure and read status from FPU are available for compatible devices.\hypertarget{group__system_autotoc_md112}{}\doxysubsubsubsection{\texorpdfstring{System Init}{System Init}}\label{group__system_autotoc_md112}
