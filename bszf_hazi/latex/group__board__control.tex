Disable a sensor. 

\begin{DoxyWarning}{Warning}
On boards 4166A, 4184A, and 4184B sensors
\begin{DoxyItemize}
\item Pressure Sensor, RH/\+Temp Sensor, and UV/\+Ambient Light Sensor;
\item UV/\+Ambient Light Sensor, Hall-\/effect Sensor, and RH/\+Temp Sensor;
\item Ambient Light Sensor, Hall-\/effect Sensor, and RH/\+Temp Sensor respectively, are tied to the same enable pin. Calling the enable function for only one of these sensors has the side-\/effect of enabling all three; and calling the disable function for only one of them has the side-\/effect of disabling all three. The latter scenario seems less than desirable.
\end{DoxyItemize}
\end{DoxyWarning}
