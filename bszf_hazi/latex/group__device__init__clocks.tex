\doxysubsubsection{Detailed Description}
Initialize the clock tree. 

Automatically configures the clock tree to use the appropriate high-\/ and low-\/frequency clock sources depending on which other device initialization components are present.\hypertarget{group__device__init__clocks_autotoc_md120}{}\doxysubsubsubsubsection{\texorpdfstring{High Frequency Clocks}{High Frequency Clocks}}\label{group__device__init__clocks_autotoc_md120}
If the {\bfseries{DPLL}} or {\bfseries{HFRCO}} device init components are present, the {\bfseries{HFRCO}} oscillator is used as the clock source for high frequency clock trees. Otherwise, if the {\bfseries{HFXO}} device init component is present, the {\bfseries{HFXO}} oscillator is used. If no high frequency oscillator device init component is present, the {\bfseries{HFRCO}} oscillator is used with its default settings.\hypertarget{group__device__init__clocks_autotoc_md121}{}\doxysubsubsubsubsection{\texorpdfstring{Low Frequency Clocks}{Low Frequency Clocks}}\label{group__device__init__clocks_autotoc_md121}
If the {\bfseries{LFXO}} device init component is present, the {\bfseries{LFXO}} oscillator is used as the clock source for low frequency clock trees. Otherwise, the {\bfseries{LFRCO}} oscillator is used.

\begin{DoxyNote}{Note}
This module has a limited configurability. Users can override the code based on the project ask. 
\end{DoxyNote}
