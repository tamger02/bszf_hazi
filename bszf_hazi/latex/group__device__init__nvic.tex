\doxysubsubsection{Detailed Description}
Initialize the NVIC Interrupt Handlers priority. 

Set all interrupt priorities, except the Fault exceptions, to {\ttfamily CORE\+\_\+\+INTERRUPT\+\_\+\+DEFAULT\+\_\+\+PRIORITY} value.

When the Device Init NVIC module is present and no configuration overwrites the em\+\_\+core {\ttfamily CORE\+\_\+\+ATOMIC\+\_\+\+METHOD} define, the atomic method is automatically set to BASEPRI ({\ttfamily CORE\+\_\+\+ATOMIC\+\_\+\+METHOD\+\_\+\+BASEPRI}).
